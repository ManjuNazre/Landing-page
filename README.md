# 📈 My Stock Assistant — Landing Page

A fully responsive, modern landing page for **My Stock Assistant** — an AI-driven stock market prediction platform. Built with **pure HTML, CSS, and JavaScript**. No frameworks. No dependencies. Just paste and run.

## 🚀 Live Preview

Open `index.html` directly in any browser — works instantly with no setup.

## ✨ Features

- Sticky navigation with smooth scrolling
- Animated live stock ticker strip (AAPL, TSLA, NVDA, and more)
- Hero section with gradient headings and CTA buttons
- Stats bar — accuracy rate, active investors, portfolio value managed
- 9 feature cards covering AI predictions, risk analysis, sentiment engine, and more
- "How It Works" 4-step process section
- Client testimonial — Mr. John Harrison · ⭐ 4.9 / 5.0 rating
- Contact section with office details, email, phone, and message form
- Fully responsive for desktop and mobile
- Dark financial theme with green accent colors
